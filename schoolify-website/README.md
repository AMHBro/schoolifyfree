# Schoolify Website

A modern React landing page for Schoolify - the comprehensive school management system.

## Features

- **Hero Section** - Engaging introduction with interactive dashboard demo
- **Features Showcase** - AI-powered analytics, privacy-first design, and user-friendly interface
- **Advanced Features Grid** - Comprehensive overview of all system capabilities
- **Responsive Design** - Optimized for all devices and screen sizes
- **Modern UI** - Dark theme with gradients and smooth animations

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS3 with modern features (Grid, Flexbox, CSS Variables)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## About Schoolify

Schoolify is a comprehensive school management system that includes:

- **Central Dashboard** - Admin panel for school management
- **Teacher Mobile App** - Flutter app for attendance and scheduling
- **Student App** - Student portal for accessing information
- **Backend API** - Secure multi-tenant backend system

The system features school code authentication, advanced analytics, and privacy-first design to ensure student data security.
